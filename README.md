### 📄 Resume / Candidate Screening System using Machine Learning

### 📌 Project Overview

This project was developed as part of Future Interns – Machine Learning Task 3 (2026).

The objective is to build a Machine Learning-based Resume Screening and Candidate Ranking System that compares resumes with a job description, calculates similarity scores, identifies missing skills, and ranks candidates based on role fit.

### 🎯 Objectives

* Load and process resume text data

* Extract relevant information from resumes

* Convert text into numerical features using TF-IDF

* Compare resumes with a job description

* Calculate similarity scores

* Rank candidates based on relevance

* Identify missing skills

### 🛠️ Technologies Used

* Python

* Google Colab

* Pandas

* Scikit-learn

* Matplotlib

### 📂 Dataset

The project uses a Resume Dataset containing multiple resumes categorized into different job domains.

Dataset Source:

[https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset](https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset)

### ⚙️ Machine Learning Workflow

1. Upload Resume Dataset

2. Load Dataset

3. Select Resume Text Column

4. Convert Resume Text using TF-IDF

5. Provide Job Description

6. Compute Cosine Similarity

7. Rank Candidates

8. Visualize Top Candidates

9. Identify Missing Skills

### 📊 Output
